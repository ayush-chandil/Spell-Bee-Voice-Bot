QUICK START INSTRUCTIONS
================================================================================

1. INSTALL DEPENDENCIES
   pip install -r requirements.txt

2. CONFIGURE ENVIRONMENT
   cp .env.example .env
   # Edit .env with your API keys:
   # DEEPGRAM_API_KEY=your_key
   # GROQ_API_KEY=your_key

3. VALIDATE SETUP
   python validate_setup.py

4. RUN SERVER
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

5. ACCESS API
   - Swagger UI: http://localhost:8000/docs
   - ReDoc: http://localhost:8000/redoc
   - Health: http://localhost:8000/health

6. RUN TESTS
   pytest tests/ -v

================================================================================
API OVERVIEW
================================================================================

REST ENDPOINTS:

POST /session/start
  Request: {"difficulty": "easy|medium|hard"}
  Response: {session_id, message, state}

GET /session/{session_id}/state
  Response: {session_id, state}

POST /session/{session_id}/next-word
  Response: {session_id, message, state}

POST /session/{session_id}/evaluate
  Request: {"is_correct": bool, "spoken_text": str}
  Response: {session_id, message, state}

POST /session/{session_id}/end
  Response: {session_id, message, state}

GET /health
  Response: {status, active_sessions}

WEBSOCKET ENDPOINT:

WS /ws/{session_id}
  Message Types: spell, next, state
  Protocol: JSON-based frame communication
